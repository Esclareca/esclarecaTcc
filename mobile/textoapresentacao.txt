>SOLUCAO:
De acordo com os problemas que evidenciamos, propomos nosso aplicativo, como solução:

através do modelo de rede social, ele irá conter:
-feed de dúvidas, para que os usuários possam fazer perguntas e discutir suas dúvidas sobre um determinado tema,
-feed de conteúdo, em que o usuário pode visualizar e recompartilhar conteúdo que seja pertinente as suas áreas de conhecimento,
através de um sistema de etiquetas que apresenta o melhor conteúdo e dúvidas relacionadas ao perfil do usuário
-tudo isso de forma interativa com os demais usuários
-através de avaliações e respostas, tornando o processo mais agradável e dinâmico
-de forma que a gamificação motive a participação do usuário


>SOBRE AS PRINCIPAIS TECNOLOGIAS:
A linguagem que utilizada foi o javascript, permitindo uma stack com uma única linguagem
 tendo o react native como biblioteca de interfaces
 o node.js como framework de backend
 o banco de dados até o momento é o sqlite
 a ide utilizada é o visual code
 e utilizamos o github como repositório do nosso projeto
